# Contexto
Eres un desarrollador web experto en Frontend y diseño UI/UX. Tu objetivo es construir la landing page para "Relicware", una empresa de desarrollo de software que ofrece servicios de creación de páginas web y aplicaciones a medida.

# Estilo Visual
El diseño debe basarse fuertemente en la temática del videojuego **Cyberpunk 2077**.
- **Colores:** Usa principalmente amarillo neón (el característico amarillo Cyberpunk), negro profundo, acentos en rojo intenso y azul cyan brillante (glitch effect).
- **Tipografía:** Usa fuentes de aspecto futurista, sans-serif gruesas y geométricas.
- **Formas y Elementos:** Emplea contenedores angulares, bordes asimétricos o recortados, botones con efectos de hover tecnológicos (como glitch o scanlines), e iconografía futurista.
- **Profesionalismo:** A pesar del fuerte tema visual Cyberpunk, el contenido y la estructura deben comunicar seriedad, experiencia técnica y confianza empresarial.

# Tareas
1. **Estructura (HTML/React):** Crea componentes para una cabecera (Navbar), Hero Section (con el título "Relicware" y un call-to-action impactante), una sección de Servicios (Web y Aplicaciones a medida), una sección de "Por qué nosotros/Beneficios", y un Footer técnico.
2. **Estilos (CSS/Tailwind):** Aplica la paleta de colores y los efectos requeridos. Añade animaciones sutiles tipo "glitch" o bordes biselados usando CSS.
3. **Contenido:** Redacta textos comerciales persuasivos y enfocados a tecnología y futuro ("Forjando el software del mañana", "Tu infraestructura digital, a prueba del futuro", etc.).

Ejecuta este prompt proporcionando el diseño y el código de cada sección para la landing page.
